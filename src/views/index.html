Image search microservice API

A freeCodeCamp challenge project

Made for this freeCodeCamp challenge. Provides an abstraction layer on top of the Qwant API, creating a microservice for easier use.

Uses Node with Express, with Ava and Supertest for testing. HTML presentation using the Bulma framework.

Source code available on Github under a Apache-2.0 license.

API usage

Search:

/api/imagesearch/{keyword}[?offset={number}]Demo

200 Returns an array of a maximum of 10 search results for {keyword}, offset by {number}. 

[
	{
		"url": "string",
		"thumbnail": "string",
		"description": "string",
		"context": "string"
	}, ...
]

400 Returns an error property on any potential errors. 

{
	"error": "string"
}

Search history:

/api/latest/Demo

200 Returns an array of a maximum of 10 search history entries. 

[
	{
		"searchKeyword": "string",
		"date": "2017-09-30T02:04:42.224Z" // ISO 8601 format
	}, ...
]

400 Returns an error property on any potential errors. 

{
	"error": "string"
}
